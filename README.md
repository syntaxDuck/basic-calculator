# Calculator

A desktop calculator application built with Python and tkinter.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division
- GUI interface with dark theme
- Clear button to reset the display
- Supports decimal numbers
- Results display up to 4 decimal places

## Requirements

- Python 3.x
- tkinter (included with Python standard library)

## Running the Calculator

```bash
python calculator.py
```

## Usage

1. Click number buttons to enter values
2. Use +, -, *, / for arithmetic operations
3. Press = to calculate the result
4. Press clear to reset the display
